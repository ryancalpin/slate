Specialty Plugin Packs

Each pack installs a set of specialty-specific mini-modules + pre-built template layouts. Mix and match freely on the canvas. Which specialties should ship first-party with the app?

❤️

Cardiology

GDMT tracker (BB/ACEi/ARNI/MRA/SGLT2i doses+targets), EF/Echo, diuresis response, hemodynamics (CI/PCWP/SVR), rhythm+pacemaker, CHADS₂-VASc, HAS-BLED, TIMI, GRACE, LVAD settings

🫁

Pulmonology / Critical Care

Vent settings (mode/FiO₂/PEEP/TV/P-plat), ARDSnet protocol, ABG interpreter, bronchodilator tracker, CURB-65, PSI, Berlin ARDS criteria, P/F ratio, weaning readiness

🫘

Nephrology

Dialysis settings (HD/PD, UF goal, access), GFR trends (CKD-EPI cited), electrolyte tracker, RRT timeline, urine studies, AKI staging (KDIGO), FENa/FEUrea

🧠

Neurology / Neurocritical Care

NIHSS (full scored), mRS, GCS, pupil tracker, ICP/CPP monitor, seizure log, stroke timeline (LKW/door-to-needle), Hunt-Hess, Fisher grade, EVD settings

🦠

Infectious Disease

Antibiotic tracker (agent/dose/day/planned duration), culture+sensitivity log, fever curve, source control tracker, Sepsis-3/qSOFA, ANC tracker, antifungal/antiviral log

🏥

ICU / Critical Care

Vasopressor tracker (multiple agents, doses, MAP target), RASS/CPOT, SAT/SBT readiness, nutrition (EN/PN, kcal/protein targets), SOFA, APACHE II, VTE prophylaxis status

🩸

Hematology / Oncology

Chemo regimen tracker (cycle/day/agents), ANC/CBC trends, transfusion log (pRBC/FFP/PLT/cryo with reactions), tumor markers, neutropenic fever protocol, dose modifications

🫀

Hepatology / GI

MELD-Na (cited), Child-Pugh, ascites tracker (paracentesis log, SBP albumin replacement), West Haven encephalopathy, lactulose/rifaximin, GI bleed (Glasgow-Blatchford, Rockall)

⚗️

Endocrinology

Insulin infusion protocol tracker, glucose log (with time-in-range), DKA tracker (anion gap, bicarb trend, closure criteria), thyroid/adrenal labs, steroid taper log

🔪

General Surgery / Post-Op

Surgical drains (output type/color/volume), wound assessment (VAC settings, dehiscence), post-op checklist, diet advancement, DVT prophylaxis, ostomy output

🤰

OB/GYN

Antepartum tracking (GA, FHR, contraction pattern), postpartum assessment, preeclampsia severity tracker (BP log, proteinuria, labs), mag drip settings

👶

Pediatrics / Neonatology

Weight-based dosing calculator, growth charts, Apgar, NAS scoring (Finnegan), NICU flowsheet (TPN components, UAC/UVC), age-adjusted vitals normals
